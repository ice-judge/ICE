# 문제해더

| 필드 | 설명 | 값 |
| --- | --- | --- |
| title | 제목 | string |
| id | 아이디 | uint |
| statistics | 시도 관련 통계 | SolveStatistics |
| difficulty | 난이도 | string |
| tags | 태그들 | string[] |

## SolveStatistics

| 필드 | 설명 | 값 |
| --- | --- | --- |
| solved | 풀린 횟수 | uint |
| total | 합계 | uint |
| rate | 해결률 | uint(0~100) |
